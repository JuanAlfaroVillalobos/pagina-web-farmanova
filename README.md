# 💊 Farmanova S.A.C. - Página Web

> **Tu salud, a un clic.**

Proyecto desarrollado para el curso de **Taller de Programación Web**, cuyo objetivo es digitalizar una farmacia comunitaria mediante el desarrollo de una página web moderna, intuitiva y adaptable a cualquier dispositivo.

---

## 📖 Descripción

Farmanova S.A.C. nace con el propósito de acercar servicios de salud y bienestar a la comunidad. Este proyecto representa la evolución digital de la farmacia, ofreciendo una plataforma web que mejora la experiencia del usuario y fortalece la presencia digital del negocio. :contentReference[oaicite:1]{index=1}

---

## 🎯 Objetivos

- Digitalizar la farmacia mediante una plataforma web.
- Mejorar la experiencia del cliente.
- Mostrar un catálogo organizado de productos.
- Incrementar la presencia digital del negocio.
- Implementar una interfaz moderna y responsiva.

---

## 🚀 Tecnologías Utilizadas

- HTML5
- CSS3
- JavaScript
- Google Fonts (Poppins)
- Font Awesome
- Flexbox
- CSS Grid

## ✨ Características

### 🎨 Diseño moderno

- Tipografía Poppins.
- Iconografía mediante Font Awesome.
- Diseño limpio y profesional.
- Organización mediante Flexbox y CSS Grid. :contentReference[oaicite:2]{index=2}

### 🖼️ Interfaz visual

- Tarjetas con sombras (Box Shadow).
- Imágenes uniformes usando `object-fit: cover`.
- Fondo con efecto Parallax (`background-attachment: fixed`). :contentReference[oaicite:3]{index=3}

### 🎬 Animaciones

- Animaciones con `@keyframes`.
- Transiciones suaves (`transition`).
- Animaciones de entrada.
- Efectos hover sobre tarjetas e imágenes.
- Cambios visuales en botones y enlaces. :contentReference[oaicite:4]{index=4}

### 📱 Diseño Responsivo

El sitio se adapta automáticamente a diferentes tamaños de pantalla:

- 📱 Móviles
- 📱 Tablets
- 💻 Laptops
- 🖥️ Escritorio

Utilizando **Media Queries** y unidades relativas (`rem`) para mejorar la accesibilidad y escalabilidad. :contentReference[oaicite:5]{index=5}

## 🎯 Problema que resuelve

Antes del desarrollo del sistema se identificaron las siguientes dificultades:

- Inventario desorganizado.
- Procesos manuales.
- Ausencia de presencia digital.
- Baja competitividad frente a grandes cadenas farmacéuticas.

La plataforma web busca mejorar la experiencia del cliente y fortalecer la presencia digital de Farmanova. :contentReference[oaicite:6]{index=6}
